# Loop Studio · 精品短剧剧本创作引擎

> 1人 + 15个AI专家 = 完整短剧制作团队

## 🎬 项目简介

Loop Studio 是一款面向独立创作者的AI精品短剧剧本创作引擎。通过策划师、人设师、节奏师、对白师、文化顾问、审核员 6 位 AI 专家协同工作，将一个简单的想法转化为 24-30 集完整短剧剧本。

## ✨ 核心特性

- **六步创作流程**：创意策划 → 人物塑造 → 节奏控制 → 对白润色 → 文化植入 → 合规审查
- **AI专家协同**：6位不同领域的AI专家各司其职，专业度更高
- **四大创作风格**：情感流💧 / 节奏党⚡ / 文化向🏛️ / 反转王🌀
- **非遗文化加持**：内置中华优秀传统文化知识库V1.2，确保文化内容准确
- **平台级合规**：AI审核员多维检测，过审率 > 92%

## 🚀 在线演示

[👉 点击进入演示](https://steady-tarsier-bf09f0.netlify.app)

输入任意短剧想法，体验从灵感到完整剧本的六步创作全流程。

## 🛠️ 技术架构

- 前端：原生 HTML/CSS/JS（演示界面）
- 引擎核心：Python + LangChain + 多Agent协同
- 文化库：中华优秀传统文化知识库 V1.2（结构化语料 10,000+ 条）
- 部署：Netlify Drop（演示界面）

## 🎯 演示内容说明

演示界面包含 6 个创作阶段，每个阶段对应一位 AI 专家的输出：

| 步骤 | 专家 | 职能 | 代表色 |
|------|------|------|--------|
| 1 | 策划师 | 选题定位、核心卖点 | 紫 |
| 2 | 人设师 | 角色设定、人物关系 | 蓝 |
| 3 | 节奏师 | 分集大纲、爽点分布 | 青 |
| 4 | 对白师 | 场景描写、台词打磨 | 绿 |
| 5 | 文化顾问 | 非遗元素、文化植入 | 金 |
| 6 | 审核员 | 合规检测、风险预警 | 粉 |

演示内容以《蓝白吟》为样本，完整展示非遗题材短剧从 0 到 1 的创作全过程。
