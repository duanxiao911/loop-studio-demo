## 期待感引擎

观众追剧的核心驱动力不是"好看"，是"想知道后面会怎样"。期待感不是感觉，是工程——可以被设计、追踪、优化。

### 信息差矩阵

信息差 = 观众和角色之间的认知落差。三种信息差制造三种不同的心理张力：

| 类型 | 定义 | 观众心理 | 使用场景 |
|------|------|----------|----------|
| 焦虑型信息差 | 观众知道但角色不知道 | "你快醒醒啊！别上当！" | 悬疑、危机、误会 |
| 悬念型信息差 | 角色知道但观众不知道 | "他到底藏着什么秘密？" | 揭秘、反转、身份 |
| 共同探索型信息差 | 观众和角色都不知道 | "到底是怎么回事？" | 冒险、成长、世界观 |

#### 信息差设计规则

1. 全剧至少维护3条信息差线（每条线跨越多集）
2. 焦虑型信息差每5集至少闭合1条（否则观众疲惫）
3. 悬念型信息差可以长线保留，但每10集必须给一次"部分揭示"作为奖励
4. 信息差的闭合本身就是剧情高潮——闭合的瞬间就是观众情绪的释放点


### 承诺-延迟-兑现循环

每一集都在给观众做"承诺"，承诺不兑现观众就弃剧，兑现太快观众不过瘾。

```
承诺（暗示将要发生的好事/坏事）
  → 延迟（制造障碍，让观众等一等）
    → 兑现（给出超出预期的回报）
```

#### 循环周期

- 微循环（1-2集）：小承诺小兑现（暗示→短暂延迟→小幅满足）
- 中循环（3-5集）：中等承诺中等兑现（重要线索出现→多方阻碍→超出预期的揭示）
- 大循环（全剧）：终极承诺终极兑现（故事前提中的核心问题→全剧的挣扎→结局的回报）

#### 承诺的四种形式

| 形式 | 示例 |
|------|------|
| 视觉暗示 | 镜头闪过一把刀（暗示即将有暴力事件） |
| 对白暗示 | "你以为事情到此结束了？"（暗示更大的危机） |
| 行为暗示 | 角色开始囤积物资（暗示即将有灾难） |
| 结构暗示 | 闪前一个画面但不解释（让观众猜一整季） |

#### 延迟的手法

- 制造障碍：角色想揭示真相但被阻止
- 转移注意：用B故事的甜蜜暂时掩盖A故事的危机
- 假兑现：看似要兑现了，结果是假的（但假兑现不能滥用，全剧不超过2次）
- 提高赌注：兑现的条件变得更难达成

## 伏笔回收系统

伏笔是观众和编剧之间的智力游戏。埋得好、收得巧，观众会有"原来如此！"的智力愉悦感。

### 伏笔三原则

1. **必收原则**：每埋一个伏笔必须在10集内回收。超过10集的伏笔必须在埋入时给观众"这是一个长期线索"的信号
2. **增值原则**：伏笔回收时，观众对原始场景的记忆必须产生新的含义——不是简单的"哦原来是这个"，而是"原来那个画面是在暗示这个！"
3. **双层回收原则**：至少30%的伏笔要回收两次——第一次回收表面含义（"原来是他干的"），第二次回收深层含义（"原来他做这件事是因为……"）

### 伏笔设计五步法

1. **选定时机**：在前1/3的剧集中埋入（观众注意力最集中的阶段）
2. **伪装嵌入**：伏笔不能看起来像伏笔——必须伪装成日常细节、环境描写、角色习惯
3. **标记类型**：道具伏笔（一个物件反复出现）、台词伏笔（一句看似普通的话后来成真）、行为伏笔（角色的一个习惯后来被揭示原因）
4. **设计回收**：回收时刻必须有情感冲击力——不能只是逻辑上的闭合
5. **设计增值**：回收后观众回看原场景时，要能发现新的信息

### 伏笔追踪表格式

```
| 伏笔内容 | 伏笔类型 | 埋入集数 | 埋入场景描述 | 回收集数 | 回收方式 | 观众重新理解 | 是否双层回收 |
|---------|---------|---------|-------------|---------|---------|-------------|------------|
```

### 伏笔密度控制

- 前10集：每2集至少埋入1个伏笔
- 中10集：伏笔回收高峰期，每2集至少回收1个
- 后10集：大伏笔的终极回收，同时可以埋入为续集/番外准备的长线伏笔
